Avatar: The Last Airbender

This is one of my favorite shows! This page also dicusses The Legend of Korra(LOK), because LOK takes place a few decades later, in the same world, and has some of the same characters.

What Element Are You?

Master Your Element!

Elements:

Waterbending

If you have waterbending, then you can control water and ice. A few people can bloodbend as well.

Examples:

  • Katara
  • Hama
  • Princess Yue

Earthbending

If you have earthbending, then you can move land. Some can also control metal and lava.

Examples:

  • Toph
  • Bumi
  • Bolin

Firebending

If you have firebending, then you can launch fire. A few people can also harness lightning. If you're a firebender, you're most likely a villain.

Examples:

  • Prince Zuko
  • Princess Azula
  • Uncle Iroh

Airbending

If you have airbending, then you can control air. Some can go through a complicated ritual and learn to fly by themselves. If you're an airbender, then you're most likely a monk and dead because of the firebenders.

Examples:

  • Aang
  • Monk Gyatso
  • Appa

Secondary Forms of Bending:

Bloodbending

If you're a waterbender, you have a chance at becoming a bloodbender. Bloodbending makes it possible to control peoples' body through their blood. It was invented by waterbender Hama to escape the imprisonment from firebenders. It is very painful to be under control of bloodbending. It was made illegal by Katara.

Example:

  • Hama
  • Katara
  • Tarrlok

Metalbending

If you're a metalbender, then you can bend metal. You have to be an earthbender to do this. This was invented by Toph to escape her capture by her parents.

Examples:

  • Toph
  • Toph's daughters
  • Korra

Lavabending

If you're a lavabender, then you can bend lava. You have to be an earthbender to do this.

Examples:

  • Avatar Roku
  • Avatar Kyoshi
  • Bolin

Lightning Bending

If you're a firebender, then you might be able to have lightning bending. This allows you to bend lighting, from shooting it at others to blocking yourself from an attack.

Examples:

  • Princess Azula
  • Uncle Iroh
  • Prince Zuko
